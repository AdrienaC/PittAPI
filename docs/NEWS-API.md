# News API

### **get_news(feed)**

#### **Parameters**
  -  `feed`: News feed - can be one of ("main_news", "cssd", "news_chronicle", "news_alerts")

#### **Returns**:
Returns a list of dictionaries with parameters 'title' and 'url' of each news article from each news feed category.
